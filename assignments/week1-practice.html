1. Read and Transform CSV

2. List Issues with the Data

Identify and list out the issues present in the data. Common issues might include missing values, duplicate rows, or incorrect data types.

Hint

Example Code:

print(df.isnull().sum())
print(df.duplicated().sum())
print(df.dtypes)

3. Steps to Clean the Data

List out the steps needed to clean up the data. This might include handling missing values, removing duplicates, correcting data types, etc.

Hint

Example Steps:

  • Fill missing values with appropriate values.
  • Drop or impute missing data.
  • Remove duplicate rows.
  • Convert data types to the correct format.

4. Jupyter Notebook Cells

In your Jupyter Notebook, separate each step of the cleaning process into individual cells. This will make your process clear and organized.

Hint

Example:

# Cell 1: Read and Transform CSV
import pandas as pd
df = pd.read_csv('path_to_csv.csv')

# Cell 2: List Issues
print(df.isnull().sum())
print(df.duplicated().sum())

# Cell 3: Clean Data
df.fillna(value='Unknown', inplace=True)
df.drop_duplicates(inplace=True)
df['column_name'] = df['column_name'].astype(int)
