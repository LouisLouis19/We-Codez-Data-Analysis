Overview

This tutorial introduces NumPy, a fundamental package for scientific computing in Python.

What is NumPy?

NumPy is a library in Python that provides support for arrays and matrices, along with a collection of mathematical functions to operate on these arrays. It is essential for scientific computing and data analysis.

Installation

To install NumPy, use the following pip command:

pip install numpy

After installation, you can check the version using:

import numpy as np
print(np.__version__)

Getting Started

Importing NumPy

To use NumPy in your Python script, you need to import it:

import numpy as np

NumPy Arrays

Creating Arrays

Arrays can be created from lists or tuples:

array1 = np.array([1, 2, 3, 4])
array2 = np.array([[1, 2], [3, 4]])

You can also use built-in functions:

zeros = np.zeros((2, 2))
ones = np.ones((2, 2))
arange = np.arange(0, 10, 2)

Array Attributes

Learn about array attributes:

print(array1.shape)  # (4,)
print(array2.shape)  # (2, 2)
print(array1.size)   # 4
print(array2.dtype)  # int64

Basic Array Operations

Indexing and Slicing

Access elements and subarrays:

print(array2[0, 1])      # 2
print(array2[:, 1])       # [2, 4]

Array Arithmetic

Perform element-wise operations:

result = array1 + 2
print(result)  # [3, 4, 5, 6]

Broadcasting

Understand basic broadcasting:

array3 = np.array([[1], [2], [3]])
array4 = np.array([1, 2])
print(array3 + array4)  # Broadcasting example

Array Manipulation

Reshaping Arrays

Change the shape of arrays:

reshaped = array1.reshape((2, 2))
print(reshaped)

Flattening Arrays

Convert arrays to 1D:

flattened = reshaped.flatten()
print(flattened)

Transposing Arrays

Transpose arrays:

transposed = array2.T
print(transposed)

Statistical Operations

Descriptive Statistics

Use methods like mean, median, std, and sum:

print(np.mean(array1))
print(np.median(array1))
print(np.std(array1))
print(np.sum(array1))

Aggregation Functions

Find min, max, and argmin:

print(np.min(array1))
print(np.max(array1))
print(np.argmin(array1))
